## 🚨 故障排除指南

### 问题1: DNS解析不生效
**症状**: 域名无法访问
**解决步骤**:
1. 等待10-30分钟 (DNS传播需要时间)
2. 使用 `./测试DNS解析.sh` 验证解析
3. 检查阿里云DNS配置是否正确
4. 清除本地DNS缓存
5. 使用不同网络环境测试

### 问题2: SSL证书错误
**症状**: 浏览器显示"不安全"
**解决步骤**:
1. 等待Render自动配置证书 (最多30分钟)
2. 在Render控制台检查证书状态
3. 确保DNS解析已生效
4. 尝试手动续期证书

### 问题3: 前端无法连接API
**症状**: API调用失败
**解决步骤**:
1. 检查 `VITE_API_BASE_URL` 配置
2. 验证 `api.c2cmarket.store` 解析正确
3. 检查后端CORS配置
4. 查看浏览器控制台错误信息

### 问题4: 应用功能异常
**症状**: 某些功能无法使用
**解决步骤**:
1. 检查浏览器控制台错误
2. 查看Render部署日志
3. 验证数据库连接正常
4. 测试API端点直接访问
